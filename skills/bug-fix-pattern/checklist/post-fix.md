# 수정 후 체크리스트

## 필수 검증 항목

### 1. 타입 체크
```bash
yarn type-check
```
- [ ] 타입 에러 없음
- [ ] 새로운 타입 에러 발생 안 함

### 2. 테스트 실행
```bash
# 관련 파일 테스트
yarn test 파일명.test.tsx

# Critical 테스트
yarn test:critical
```
- [ ] 기존 테스트 통과
- [ ] 새로운 테스트 실패 없음

### 3. 코드 품질
- [ ] 기존 코드 스타일 유지
- [ ] 불필요한 공백/줄바꿈 변경 없음
- [ ] Import 문 정리
- [ ] 사용하지 않는 변수 없음

### 4. 사이드 이펙트 확인
- [ ] 다른 파일에 영향 없음
- [ ] Props 인터페이스 변경 없음
- [ ] 전역 상태 변경 없음

### 5. 문서화
- [ ] 주석이 필요한 경우 추가
- [ ] 복잡한 로직은 설명 추가

## 보고 형식

```markdown
## 수정 완료

### 변경 파일
- src/path/to/file.tsx

### 변경 내용
- 간략한 설명

### 검증 결과
✅ 타입 체크 통과
✅ 테스트 통과
✅ 사이드 이펙트 없음
```

# 📊 실시간 메트릭스 대시보드

워크플로우 실행 통계와 성능 지표를 실시간으로 표시합니다.

## 사용법

```bash
/dashboard [옵션]
```

## 옵션

- **없음**: 현재 세션의 실시간 메트릭 표시
- `--today`: 오늘의 누적 통계
- `--summary`: 전체 누적 통계
- `--reset`: 현재 세션 메트릭 초기화

## 표시 항목

### 📊 토큰 사용량
- 총 토큰 사용량
- 캐시를 통한 토큰 절감량
- 절감 효율성 (%)

### ⚡ 성능 메트릭
- 워크플로우 실행 시간
- 캐시 히트율 (%)
- 병렬 처리 에이전트 수

### ✅ 품질 지표
- 테스트 커버리지 (%)
- 타입 체크 통과 여부
- 린트 통과 여부
- 보안 이슈 개수

### 🎯 생산성 지표
- 완료된 작업 수
- 수정된 버그 수
- 추가된 기능 수
- 변경된 파일 수
- 추가/삭제된 코드 라인 수

### 🔧 Git 상태
- 현재 브랜치
- 오늘의 커밋 수
- 커밋되지 않은 변경사항

## 실행

이 명령어를 실행하면 대시보드 생성기가 실행됩니다.

**주의**: 이 명령어는 읽기 전용이며 프로젝트를 수정하지 않습니다.

---

## Implementation

대시보드를 생성하고 표시합니다.

**실행 단계**:

### Step 1: 대시보드 생성기 실행

대시보드 생성 스크립트를 초기화합니다.

### Step 2: 현재 메트릭 수집 및 표시

메트릭 데이터를 수집하고 포맷팅합니다:
- 토큰 사용량 계산
- 성능 지표 측정
- 품질 지표 확인
- Git 상태 조회

### Step 3: 사용자에게 결과 반환

포맷팅된 대시보드를 터미널에 출력합니다.

**사용 예시**:

```bash
# 현재 세션 대시보드
/dashboard

# 오늘의 통계
/dashboard --today

# 전체 누적 통계
/dashboard --summary

# 메트릭 초기화 (새로 시작)
/dashboard --reset
```

---

이제 대시보드를 실행하겠습니다.

사용자가 제공한 옵션에 따라 적절한 대시보드를 표시합니다:
- 옵션이 없으면 현재 세션 메트릭을 표시
- `--today` 옵션이면 오늘의 통계를 표시
- `--summary` 옵션이면 전체 누적 통계를 표시
- `--reset` 옵션이면 메트릭을 초기화

대시보드 스크립트를 실행하여 결과를 표시합니다.

# /major-clarify - Clarification 질문 (Step 2/4, 선택사항)

spec.md의 모호한 부분을 해결하기 위한 타겟팅된 질문을 생성합니다.

## 사용법

```
/major-clarify [feature-number]
```

예시:
```
/major-clarify 001
```

## 실행 내용

`/major`의 4단계를 독립적으로 실행:
1. spec.md 분석
2. 최대 5개의 고도로 타겟팅된 질문 생성
3. 답변을 spec.md에 통합

## 질문 우선순위

1. Critical Path 불명확성
2. 데이터 모델 모호성
3. API 계약 불명확
4. 에러 처리 전략
5. 성능 요구사항
